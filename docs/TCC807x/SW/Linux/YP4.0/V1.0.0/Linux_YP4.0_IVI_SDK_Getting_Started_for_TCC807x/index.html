note

For more details, refer to the following document [8]:

  • TCC803x Common Hardware-Assembly Manual for EVB
important

The default LCD setting of Linux_YP4.0_IVI is fitted with "TCCXXXX_AUO_WLCD_12.3". To use “TCC80XX_BOE_WLCD_12.3”, modify the build configuration as described in Chapter 4.6.3.5.5.
